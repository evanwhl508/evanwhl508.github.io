# evanwhl508.github.io

Developer website and privacy policy hosting for my mobile apps.

## Setup

This repo is configured as a GitHub Pages site. It auto-deploys from the `main` branch.

**Live URL:** https://evanwhl508.github.io

## Structure

```
index.html                  # Developer landing page
app-ads.txt                 # AdMob authorized sellers (required for ad revenue)
mj-actuary/
  privacy.html              # MJ Actuary privacy policy
```

## Before Publishing

1. Create the Gmail account `mjactuary.dev@gmail.com` (or update the email in all files)
2. Replace `pub-XXXXXXXXXXXXXXXX` in `app-ads.txt` with your actual AdMob publisher ID
3. Push to GitHub — the site goes live automatically
